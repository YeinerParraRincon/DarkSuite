# 💀 DarkSuite

> Toolkit de Ciberseguridad en Python (En desarrollo 🚧)

---

## 🧠 Acerca del proyecto

**DarkSuite** es una suite personal de herramientas de ciberseguridad desarrolladas en Python, enfocadas en el aprendizaje práctico de:

- 🔍 OSINT (Open Source Intelligence)  
- 🌐 Redes y escaneo de puertos  
- 🔐 Cracking de hashes  

Este proyecto forma parte de mi proceso de formación en desarrollo backend y ciberseguridad ofensiva.

---

## ⚙️ Módulos actuales

### 💀 DarkHash  
Herramienta para el cracking de hashes y análisis de contraseñas.

### 🔍 DarkFinder  
Herramienta OSINT para buscar nombres de usuario en múltiples plataformas.

### 🗺️ DarkMap  
Escáner de puertos TCP con multithreading para análisis de redes.
